# Security Policy  
# 安全策略

## Supported Versions  
## 支持的版本

MatterSim-WebUI is under active development.  
本项目处于持续开发中。  
Security fixes apply to the latest version only.  
安全修复仅适用于最新版本。

---

## Reporting a Vulnerability  
## 报告安全漏洞

 **Do NOT open a public issue with sensitive details.**  
 **请不要在公开 Issue 中透露敏感信息。**

Instead, please:  
请改为：

- Open a minimal Issue without details  
  提交一个不包含敏感细节的 Issue  
- Or contact the maintainer via GitHub email  
  或通过 GitHub 邮箱私下联系维护者

Please include:  
请提供：

- Description of the vulnerability  
  漏洞描述  
- Steps to reproduce  
  复现步骤  
- Potential impact  
  潜在影响  
- Suggested fix (optional)  
  修复建议（可选）

We will respond as soon as possible.  
我们会尽快回复。
