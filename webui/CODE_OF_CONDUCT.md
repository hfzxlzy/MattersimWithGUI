# Code of Conduct  
# 行为准则

## Our Pledge  
## 我们的承诺

We aim to provide a welcoming, harassment‑free environment for everyone.  
我们致力于提供一个友好、无骚扰的社区环境。

---

## Our Standards  
## 我们的标准

### Positive behavior includes:  
### 积极行为包括：

- Respectful and inclusive language  
  使用尊重、包容的语言  
- Considering different viewpoints  
  尊重不同观点  
- Constructive feedback  
  提供建设性反馈  
- Empathy toward others  
  对他人保持同理心  

### Unacceptable behavior includes:  
### 不可接受的行为包括：

- Harassment or discrimination  
  骚扰或歧视  
- Personal attacks  
  人身攻击  
- Publishing private information  
  公开他人隐私  
- Any inappropriate conduct  
  任何不当行为

---

## Enforcement  
## 执行

Project maintainers may remove comments, commits, or contributions that violate this Code of Conduct.  
维护者有权删除违反本准则的评论、提交或贡献。

If you witness or experience unacceptable behavior, please report it via GitHub Issues.  
如遇不当行为，请通过 GitHub Issues 反馈。

---

## Scope  
## 适用范围

This Code applies to all project spaces and public interactions related to the project.  
本准则适用于所有项目空间及相关公共互动。
